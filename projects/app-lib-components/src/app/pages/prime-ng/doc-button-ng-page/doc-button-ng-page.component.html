Button NG Component

Componente de botón basado en PrimeNG con funcionalidades avanzadas como estados de carga, mensajes rotativos, tooltips y múltiples variantes de estilo.

Importación

import {{ '{' }} ButtonNgComponent {{ '}' }} from 'lib-components';

Uso Básico

El componente requiere una configuración de tipo IButtonConfig que se pasa mediante el input buttonConfig.

Botón Básico

{{ '{' }}
  label: 'Click Me',
  severity: 'secondary'
{{ '}' }}

Severidades

El componente soporta diferentes severidades que cambian el color del botón.

Con Iconos

Puedes agregar iconos de PrimeIcons al botón. El icono puede posicionarse a la izquierda, derecha, arriba o abajo del texto.

Variantes

Diferentes estilos de botón disponibles para adaptarse a diversos diseños.

Solo muestra el borde sin fondo

Text

Rounded

Raised

Efecto de elevación con sombra

Badge

Agrega un badge al botón para mostrar notificaciones o contadores. Útil para indicar cantidad de elementos pendientes.

Full Width

Interfaz IButtonConfig

Referencia completa de todas las propiedades disponibles en la configuración del botón.

Propiedad Tipo Descripción
label string? Texto que se muestra en el botón
icon string? Icono de PrimeIcons a mostrar (ej: 'pi pi-save')
severity 'secondary' | 'success' | 'info' | 'warn' | 'help' | 'danger' | 'contrast' Severidad/color del botón que define su apariencia
loading boolean? Muestra un indicador de carga y deshabilita el botón
loadingMessages string[]? Array de mensajes que rotan durante la carga (cada 6 segundos)
onClick () => void? Función que se ejecuta al hacer clic en el botón
disabled boolean? Deshabilita el botón impidiendo interacciones
rounded boolean? Aplica bordes completamente redondeados
raised boolean? Aplica un efecto de elevación con sombra
outlined boolean? Estilo outlined que muestra solo el borde sin fondo
text boolean? Estilo de texto sin fondo ni borde visible
variant 'text' | 'outlined'? Variante del botón (alternativa a las propiedades booleanas)
badge string? Texto del badge a mostrar en la esquina superior derecha
size 'small' | 'large'? Tamaño del botón (por defecto es 'medium')
tooltipConfig ITooltipConfig? Configuración del tooltip con mensaje y posición
fullWidth boolean? El botón ocupa todo el ancho disponible del contenedor
fullHeight boolean? El botón ocupa toda la altura disponible del contenedor
iconPos 'left' | 'right' | 'top' | 'bottom'? Posición del icono relativo al texto (por defecto 'left')
hasTemplate boolean? Permite usar contenido personalizado con ng-content
class string? Clases CSS adicionales para personalización
style Record<string, string>? Estilos inline adicionales para personalización avanzada
