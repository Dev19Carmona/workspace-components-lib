Table NG Component

Componente de tabla avanzado basado en PrimeNG con funcionalidades como paginación, filtros globales y por columna, selección múltiple, edición inline, lazy loading, exportación a Excel/PDF y mucho más.

Importación

import {{ '{' }} TableNgComponent {{ '}' }} from 'lib-components';
import type {{ '{' }} ITableNgConfig, ITableNgData {{ '}' }} from 'lib-components';

Uso Básico

El componente requiere un array de ITableNgData y una configuración ITableNgConfig.

<app-table-ng
  [data]="tableData"
  [config]="tableConfig"
/>

Filtro Global

Habilita un campo de búsqueda global que filtra en múltiples columnas simultáneamente.

globalFilterConfig: {{ '{' }}
  isEnabled: true,
  globalFilterFields: ['name', 'email', 'status']
{{ '}' }}

Selección Múltiple

Permite seleccionar múltiples filas con checkboxes. Incluye gestión de elementos seleccionados.

selectionTableConfig: {{ '{' }}
  isEnabled: true,
  showManagementConfig: true
{{ '}' }}

Título de Tabla

Agrega un título personalizado a la tabla.

titleConfig: {{ '{' }}
  isEnabled: true,
  title: 'Lista de Usuarios'
{{ '}' }}

Exportación de Datos

Exporta los datos de la tabla a Excel o PDF con un solo clic.

excelConfig: {{ '{' }}
  isEnabled: true,
  name: 'usuarios'
{{ '}' }},
pdfConfig: {{ '{' }}
  isEnabled: true,
  name: 'usuarios'
{{ '}' }}

Paginación

Configura la paginación de la tabla con opciones personalizables de filas por página.

paginationConfig: {{ '{' }}
  paginator: true,
  rows: 5,
  rowsPerPageOptions: [5, 10, 20]
{{ '}' }}

La paginación se muestra automáticamente en la parte inferior de la tabla.

Lazy Loading

Para tablas con grandes volúmenes de datos, puedes usar lazy loading para cargar datos bajo demanda.

lazyLoadingConfig: {{ '{' }}
  isEnabled: true,
  totalRecords: 1000
{{ '}' }}

El evento lazyLoading se emite cuando se necesita cargar más datos.

Edición Inline

Permite editar celdas o filas directamente en la tabla. Soporta edición por celda o por fila completa.

editConfig: {{ '{' }}
  isEnabled: true,
  type: 'cell' | 'row',
  inlineControls: {{ '{' }}
    // configuración de controles inline
  {{ '}' }}
{{ '}' }}

Filtros por Columna

Configura filtros específicos para cada columna con diferentes tipos y modos de coincidencia.

filterConfigByKey: {{ '{' }}
  'status': {{ '{' }}
    isEnabled: true,
    primeNgColumnFilterConfig: {{ '{' }}
      type: 'text',
      matchMode: 'contains'
    {{ '}' }}
  {{ '}' }}
{{ '}' }}

Columnas Congeladas

Congela columnas específicas para que permanezcan visibles al hacer scroll horizontal.

frozenColumnConfigByKey: {{ '{' }}
  'name': true,
  'actions': true
{{ '}' }}

Scroll Configurado

Habilita scroll vertical y horizontal con altura personalizable.

scrollConfig: {{ '{' }}
  isEnabled: true,
  scrollHeight: '400px'
{{ '}' }}

Interfaz ITableNgConfig

Referencia completa de todas las propiedades disponibles en la configuración de la tabla.

Propiedad Tipo Descripción
keys string[] Array de claves que definen las columnas a mostrar (requerido)
keysNames Record<string, string>? Mapeo de claves a nombres de columnas para mostrar en el header
paginationConfig IPaginationConfig Configuración de paginación (requerido)
globalFilterConfig IGlobalFilterConfig? Configuración del filtro global de búsqueda
selectionTableConfig ISelectionTableConfig? Configuración de selección múltiple de filas
filterConfigByKey Record<string, IFilterConfigByKey>? Configuración de filtros por columna específica
frozenColumnConfigByKey Record<string, boolean>? Configuración de columnas congeladas
scrollConfig IScrollConfig? Configuración de scroll vertical y horizontal
lazyLoadingConfig ITableNgLazyLoading? Configuración de carga perezosa de datos
columnConfig IColumnConfig? Configuración de tamaño de columnas
titleConfig ITitleConfig? Configuración del título de la tabla
selectedColumns IColumnPrimeNg[]? Columnas seleccionadas para mostrar
rowExpansionConfig IRowExpansionConfig? Configuración de expansión de filas
toolbarConfig IToolbarConfig? Configuración de la barra de herramientas
hideToolbar boolean? Oculta la barra de herramientas
excelConfig IExcelConfig? Configuración de exportación a Excel
pdfConfig IPdfConfig? Configuración de exportación a PDF

Interfaz ITableNgData

Estructura de datos para cada fila de la tabla.

Propiedad Tipo Descripción
rowData T Datos de la fila (requerido)
rowDataButtons Record<string, IButtonConfig[]>? Botones por columna específica
rowDataComponents Record<string, TemplateRef>? Componentes personalizados por columna
typeCell Record<string, ETypeInput>? Tipo de celda para edición inline
currencyTypeCellConfig Record<string, ICurrencyConfig>? Configuración de celdas de tipo moneda
componentTypeCellConfig Record<string, TemplateRef>? Configuración de componentes personalizados en celdas
raw R Datos originales sin procesar (requerido)
onClick (data?: unknown) => void Función que se ejecuta al hacer clic en la fila (requerido)
sortConfig ISortConfig? Configuración de ordenamiento personalizado
id string Identificador único de la fila (requerido)
isSelected boolean? Indica si la fila está seleccionada
isLoading boolean? Indica si la fila está en estado de carga
