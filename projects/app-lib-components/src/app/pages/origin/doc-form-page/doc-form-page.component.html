Form & Inline Input Components

Componentes de formulario avanzados con soporte para múltiples tipos de inputs, validaciones, eventos de cambio y configuración flexible. El componente FormComponent gestiona formularios completos, mientras que InlineInputComponent puede usarse individualmente.

Importación

import { FormComponent, InlineInputComponent } from 'lib-components';
import type { IControlConfig, IFormConfig } from 'lib-components';
import { ETypeInput } from 'lib-components';

Form Component - Uso Básico

El componente FormComponent requiere un FormGroup y un array de IControlConfig.

<app-form
  [formGroup]="formGroup"
  [controls]="controls"
  (formSubmit)="onFormSubmit($event)"
  (formChanges)="onFormChanges($event)"
/>

Tipos de Input Disponibles

El componente soporta múltiples tipos de inputs a través del enum ETypeInput.

ETypeInput: {
  TEXT, PASSWORD, EMAIL, NUMBER, DATE, TIME,
  DATETIME_LOCAL, SEARCH, TEL, URL, COLOR,
  MONTH, WEEK, FILE, CHECKBOX, RADIO, RANGE,
  HIDDEN, TEXTAREA, SELECT, MULTISELECT,
  AUTOCOMPLETE, SWITCH, CALENDAR, DROPDOWN,
  CHIPS, TOGGLE, OTP, CURRENCY, KNOB, BADGE,
  DECIMAL, IMAGE, COMPONENT
}

Validaciones

Puedes agregar validadores de Angular Forms a cada control. Los errores se muestran automáticamente.

control: new FormControl('', [
  Validators.required,
  Validators.email,
  Validators.minLength(5)
])

Select y MultiSelect

Configuración de selectores con opciones personalizadas.

selectConfig: {
  options: [
    { code: '1', name: 'Opción 1' },
    { code: '2', name: 'Opción 2' }
  ]
}

Currency Input

Input especializado para valores monetarios con formato automático.

typeInput: ETypeInput.CURRENCY,
currencyConfig: {
  currency: 'USD',
  locale: 'en-US'
}

Inline Input Component

El componente InlineInputComponent puede usarse individualmente sin necesidad de un FormGroup. Implementa ControlValueAccessor para integrarse con Angular Forms.

<app-inline-input
  [config]="{ typeInput: ETypeInput.TEXT, label: 'Nombre' }"
  [(value)]="textValue"
  (outputValue)="onValueChange($event)"
/>

Eventos del Form Component

El componente emite varios eventos para manejar cambios y envíos del formulario.

(formSubmit)="onFormSubmit($event)"      // Al enviar el formulario
(formChanges)="onFormChanges($event)"      // En cada cambio
(formChangesDebounced)="onDebounced($event)" // Cambios con debounce (650ms)

Eventos del InlineInput Component

El componente InlineInput emite eventos para cambios de valor.

(outputValue)="onValueChange($event)"     // En cada cambio
(outputBlur)="onBlur($event)"              // Al perder el foco
(outputDebounced)="onDebounced($event)"    // Cambios con debounce (300ms)

Configuración de Textarea

Personaliza el área de texto con múltiples opciones.

textareaConfig: {
  rows: 4,
  cols: 50,
  autoResize: true,
  resize: 'vertical',
  variant: 'outlined'
}

Configuración de Date

Configura el selector de fechas con restricciones y modos de selección.

dateConfig: {
  minDate: new Date(),
  maxDate: new Date('2025-12-31'),
  selectionMode: 'single' | 'multiple' | 'range'
}

Interfaz IControlConfig

Referencia completa de todas las propiedades disponibles para configurar controles.

Propiedad Tipo Descripción
controlName string Nombre del control en el FormGroup (requerido)
control FormControl Instancia del FormControl (requerido)
typeInput ETypeInput Tipo de input a renderizar (requerido)
label string? Etiqueta del campo
readonly boolean? Hace el campo de solo lectura
hideLabel boolean? Oculta la etiqueta del campo
colSpan number? Número de columnas que ocupa el campo en el grid
selectConfig ISelectConfig? Configuración para inputs de tipo SELECT
multiSelectConfig IMultiSelectConfig? Configuración para inputs de tipo MULTISELECT
dateConfig IDateConfig? Configuración para inputs de tipo DATE/CALENDAR
textareaConfig ITextareaConfig? Configuración para inputs de tipo TEXTAREA
currencyConfig ICurrencyConfig? Configuración para inputs de tipo CURRENCY
toggleConfig IToggleButtonConfig? Configuración para inputs de tipo TOGGLE
otpConfig object? Configuración para inputs de tipo OTP

Interfaz InlineControlConfig

Configuración específica para el componente InlineInput cuando se usa individualmente.

Propiedad Tipo Descripción
typeInput ETypeInput Tipo de input a renderizar (requerido)
label string? Etiqueta del campo
isDisabled boolean? Deshabilita el campo
required boolean? Indica si el campo es requerido
readonly boolean? Hace el campo de solo lectura
selectConfig ISelectConfig? Configuración para inputs de tipo SELECT
currencyConfig ICurrencyConfig? Configuración para inputs de tipo CURRENCY
numberConfig INumberConfig? Configuración para inputs de tipo NUMBER
knobConfig IKnobConfig? Configuración para inputs de tipo KNOB
textareaConfig ITextareaConfig? Configuración para inputs de tipo TEXTAREA
